# LLM Zoomcamp - My Learning Journey

## 📚 About

This repository contains all assignments, projects, and learning materials from the **LLM Zoomcamp** course by DataTalksClub. LLM Zoomcamp is a free online course about real-life applications of Large Language Models.

## 🎯 Course Overview

In this 10-week course, I'm learning how to build AI systems that answer questions about knowledge bases, covering:
- Agentic RAG (Retrieval-Augmented Generation)
- Vector Search and Databases
- Orchestration and Workflows
- Evaluation of LLM Systems
- Monitoring and Best Practices

## 🚀 Getting Started

### Prerequisites

* Python 3.10+
* Git
* Google Colab (optional, for GPU access)

### Installation

1. **Clone the repository:**
```bash
git clone [https://github.com/beingAnujChaudhary/LLM-Zoomcamp.git](https://github.com/beingAnujChaudhary/LLM-Zoomcamp.git)
cd LLM-Zoomcamp

```


2. **Create a virtual environment:**
```powershell
python -m venv venv
.\venv\Scripts\activate  # On Windows

```


3. **Install dependencies:**
```bash
pip install -r requirements.txt
# For heavy tools later (like mlflow, wandb, local models):
# pip install -r requirements-optional.txt

```


4. **Set up environment variables:**
Create a `.env` file in the root directory and add your necessary API keys. This file is safely ignored by Git.
